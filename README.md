## Dotfiles

### Dependencies
* `sudo apt install git curl build-essential`

### How to install
`git clone git@github.com:glacierrdev/dotfiles.git "${HOME}/.dotfiles" && cd "${HOME}/.dotfiles" && make setup`
#### NOTE:
* `oh-my-zsh` require a password to change default shell.
